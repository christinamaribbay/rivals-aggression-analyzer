<h1> Instructions for Running the Program </h1>
It is recommended to run this program in CLion as this project was created in the IDE.

To avoid any compilation errors, ensure that the CMakeLists.txt file includes the .cpp files. When running main, check if the configuration lists the .cpp files under compiler options and that the working directory is the include folder.

<h2>Working the Menu</h2>
To select an option from the menu, simply type in the number corresponding with the option and hit enter.

- Option 1: The results will automatically be given after hitting enter.
- Option 2: Enter a custom input from your keyboard to receive a calculated aggression score.
- Option 3: Same as option 1.
- Option 4: Exits and terminates the program after hitting enter.
